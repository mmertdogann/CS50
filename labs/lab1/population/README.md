Determine how long it takes for a population to reach a particular size.

Sample behaves of the program:

```
$ ./population
Start size: 100
End size: 200
Years: 9
```

```
$ ./population
Start size: -5
Start size: 3
Start size: 9
End size: 5
End size: 18
Years: 8
```

>Read the implementation details here [here](https://cs50.harvard.edu/x/2021/labs/1/population/)
